# Brazika Product Landing

Landing estática criada do zero para a Blusa Tricot Brasil - Brazilcore 2026.

## Arquivos

- `index.html`: página completa com CSS e JavaScript embutidos.

## Publicação gratuita em subdomínio

Esta página está pronta para GitHub Pages no subdomínio:

`bio.brazika.com.br`

Para usar GitHub Pages sem interferir na Shopify, mantenha `www.brazika.com.br` e o domínio raiz como estão e crie/altere apenas o CNAME do subdomínio:

```txt
Tipo: CNAME
Nome: bio
Destino: brazika.github.io
```

Se preferir Cloudflare Pages, o CNAME pode apontar para `brazika-links.pages.dev`, mas o projeto `brazika-links` precisa existir e estar publicado dentro da Cloudflare Pages.

## UTMs

Todos os CTAs e links externos da página usam:

- `utm_source=landing_brazilcore`
- `utm_campaign=blusa_tricot_brazilcore_2026`

O `utm_medium` e `utm_content` mudam conforme a posição do botão/link.
